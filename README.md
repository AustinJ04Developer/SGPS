# 🅿️ Smart Parking Monitor

Smart Parking Monitor is a real-time parking management dashboard built for urban lots equipped with IoT sensors. It visualises live slot availability on an interactive map — colour-coded by occupancy level — and updates automatically as sensors report changes, helping drivers locate free parking without guesswork. A real-time parking availability dashboard that shows live slot occupancy across multiple parking locations on an interactive map.

---

## Features

- **Live map** — interactive Google Maps view with colour-coded markers per parking lot
- **Availability at a glance** — green (available), amber (almost full), and red (full) indicators
- **Slot detail** — tap any marker to see per-slot status and occupancy counts
- **Satellite view** — toggle between map and satellite imagery
- **GPS-pending handling** — lots awaiting a GPS fix are excluded from the map with a status notice
- **Mobile friendly** — responsive layout that works on any screen size
